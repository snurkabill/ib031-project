Využité algoritmy

Zadány byly lineární modely a regresní stromy. To nás vedlo k myšlence vyzkoušet jednoduché lineární modely a regresní stromy. Samozřejmě existuje velké množství různých lineárnních modelů, některé sofistikovanější metody ale byly již zadáni jiným týmům (například SVM), tudíž jsme se rozhodli pro modely: Generalized Linear Model, Linear Discriminant Analysis, Linear Regression a Random Forest.

Balíček GLM realizoval jednoduchou lineární regresi, ovšem během našich pokusů jsme nejspíše objevili bug, protože na specifickém datasetu, který zabíral přibližně 15mb si vzal více než 30gb paměti. Z těchto důvodů jsme museli omezit výběr datasetů na menší vzorky.

Linear Discriminant Analysis: Metoda, která hledá lineární kombinaci atributů z dat, takovou, že nejlépe separuje data do tříd.

Linear model: jednoduchý, "původní", balíček pro fitování lineárních modelů pro klasifikaci nebo regresi.

Random forest: bylo vygenerováno 500 stromů, vybudovaných jako regresní model, a výsledek poté průměrován.
